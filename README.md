# Meeoto (project name)

Online platform for classic car lovers that centralize clubs, events and helps on one and unique place.

## Get started

1. Clone the project
```
git clone https://github.com/Meeoto/meeoto.git
```
2. Create a `.env` file
```
make copy-env
```
3. install dependencies
```
make composer-install
make npm-install
```
4. Start docker stack
```
make start
```
